Keyboard shortcuts

  • Tab

    Autocomplete from the cursor position

  • Home

    Move cursor to start of the line

  • End

    Move cursor to end of the line

  • Ctrl + l

    Clear the screen

  • Ctrl + r

    Search history including the current characters. Repeat for next command

  • Ctrl + u

    Cut line before cursor into clipboard

  • Ctrl + w

    Cut word before cursor into clipboard

  • Ctrl + y

    Paste from clipboard

  • Ctrl + x Ctrl + u

    Undo previous operation
