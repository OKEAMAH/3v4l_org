Based on work by Fabrice Bellard

Keyboard shortcuts

  • Tab

    Autocomplete functions and variables from the cursor position. Repeat for complete list

  • Home

    Move cursor to start of the line

  • End

    Move cursor to end of the line

  • Ctrl + l

    Clear the screen

  • Ctrl + r

    Search history, repeat for next command

  • Ctrl + d

    Restart the shell
